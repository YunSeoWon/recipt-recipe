# 👨‍👩‍👧‍👦 Recipt-api-base
Recipe + Reciept = **Recipt**

MSA 환경 요리 레시피 웹 서비스 Toy Project MSA API 모듈 베이스 레포지토리 입니다.

Recipt의 모든 MSA API 프로젝트는 이 레포지토리에서 시작합니다.

내가 프로젝트 새로 파기 귀찮기 때문에 만드는 것!
